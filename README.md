# Vulnerability Assessment Report

## 📌 Description
Performed security assessment on an e-commerce website.

## 🛠️ Tools Used
- Nmap
- OWASP ZAP
- Browser Developer Tools

## 🚨 Findings
- Missing X-Frame-Options header
- Missing X-Content-Type-Options
- Missing HSTS
- Server information disclosure

## 🎯 Outcome
Identified key security misconfigurations and suggested improvements.
